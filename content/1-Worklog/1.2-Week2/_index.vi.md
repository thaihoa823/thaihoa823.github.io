### Kết quả đạt được tuần 2:

* Hiểu vai trò của Amazon Route 53 trong việc quản lý và phân giải tên miền.

* Biết Amazon CloudFront giúp phân phối website và nội dung nhanh hơn đến người dùng.

* Phân biệt được chức năng cơ bản của:
  * Application Load Balancer
  * Network Load Balancer
  * Amazon API Gateway

* Nắm được các thành phần chính của Amazon VPC:
  * Subnet
  * Route Table
  * Internet Gateway
  * NAT Gateway

* Hiểu cách VPC Peering, Site-to-Site VPN và AWS Transit Gateway hỗ trợ kết nối các hệ thống mạng trên AWS.

* Phân biệt được Security Group và Network ACL trong việc kiểm soát lưu lượng truy cập.

* Biết AWS WAF hỗ trợ lọc và bảo vệ website trước các yêu cầu không phù hợp.

* Hiểu VPC Flow Logs dùng để ghi nhận lưu lượng mạng trong VPC.

* Làm quen với CloudWatch Metrics, Logs và Alarms để theo dõi tài nguyên AWS.

* Biết Reachability Analyzer hỗ trợ kiểm tra đường kết nối giữa các tài nguyên trong VPC.

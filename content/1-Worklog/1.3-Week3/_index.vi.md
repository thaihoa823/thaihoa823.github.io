### Kết quả đạt được tuần 3:

* Hiểu Amazon EC2 cung cấp máy chủ ảo có thể tùy chỉnh hệ điều hành và tài nguyên.

* Biết Amazon Lightsail phù hợp với những ứng dụng cần triển khai nhanh và cấu hình đơn giản.

* Phân biệt được các mô hình tính toán:
  * Máy chủ ảo với Amazon EC2
  * Container với Amazon ECS và Amazon EKS
  * Serverless với AWS Lambda

* Hiểu Amazon ECR được sử dụng để lưu trữ và quản lý Docker Image.

* Biết AWS Fargate hỗ trợ chạy container mà không cần trực tiếp quản lý máy chủ.

* Hiểu AWS Lambda có thể thực thi mã nguồn khi nhận được sự kiện từ dịch vụ khác.

* Nắm được các thành phần cơ bản của Amazon S3:
  * Bucket
  * Object
  * Object Key
  * Bucket Versioning
  * Block Public Access

* Biết cách tạo EC2 Instance, kết nối máy chủ và cấu hình Security Group cơ bản.

* Biết cách tạo S3 Bucket, tải dữ liệu lên và kiểm tra các Object đã lưu trữ.
